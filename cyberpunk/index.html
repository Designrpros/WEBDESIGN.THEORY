Design Theory 003

CYBER

PUNK

The Philosophy

High tech, low life. Neon bleeding through rain-streaked windows. The future is corrupt, beautiful, and running on borrowed power.

The Aesthetic

Glitch art. Neon grids. Scanlines. Text that pulses like it's barely holding together. Beauty in digital decay.

HISTORY

1980s

The Origins

Cyberpunk emerges from William Gibson's "Neuromancer" and Ridley Scott's "Blade Runner." The term itself coined by Bruce Sterling. Dark futures, bright lights.

1990s

Cinema & Games

"Ghost in the Shell," "Akira," "The Matrix." Cyberpunk infiltrates mainstream consciousness. Video games like "Deus Ex" define the visual language.

2010s

Digital Renaissance

Cyberpunk 2077, "Westworld," "Altered Carbon." The aesthetic evolves from analog to hyper-digital. Neon meets wireframe. Glitch becomes intentional.

TODAY

The Web Era

Cyberpunk aesthetics dominate UI/UX design. Neon glows, dark themes, glitch animations. The future we were promised, now in your browser.

COLOR PALETTE

#FFE600

Neon Yellow

Primary accent. High visibility warning color.

#00F0FF

Cyan

Secondary accent. Tech/interface color.

#FF00FF

Magenta

Tertiary accent. Glitch/alternate color.

#0D0D0D

Void Black

Background. Makes neon colors pop.

CODE EXAMPLES

CSS: Neon Glow Effect
.neon-glow {
  color: #FFE600;
  text-shadow:
    0 0 5px #FFE600,
    0 0 10px #FFE600,
    0 0 20px #FFE600,
    0 0 40px #FF8C00;
}

.box-glow {
  box-shadow:
    0 0 5px #00F0FF,
    0 0 10px #00F0FF,
    0 0 20px #00F0FF,
    inset 0 0 10px rgba(0,240,255,0.3);
}
CSS: Glitch Animation
.glitch::before,
.glitch::after {
  content: attr(data-text);
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
}

.glitch::before {
  color: #00F0FF;
  animation: glitch-1 0.3s infinite;
  clip-path: polygon(0 0, 100% 0, 100% 33%, 0 33%);
}

.glitch::after {
  color: #FF00FF;
  animation: glitch-2 0.3s infinite reverse;
  clip-path: polygon(0 67%, 100% 67%, 100% 100%, 0 100%);
}
CSS: Neon Button
.neon-btn {
  background: transparent;
  border: 2px solid #FFE600;
  color: #FFE600;
  box-shadow:
    0 0 5px #FFE600,
    0 0 10px #FFE600,
    inset 0 0 5px rgba(255,230,0,0.2);
  transition: all 0.3s ease;
}

.neon-btn:hover {
  background: #FFE600;
  color: #0D0D0D;
  box-shadow:
    0 0 20px #FFE600,
    0 0 40px #FFE600,
    0 0 60px #FFE600;
}
CSS: Animated Border
.cyberpunk-border {
  position: relative;
  background: #0D0D0D;
}

.cyberpunk-border::before {
  content: '';
  position: absolute;
  inset: -2px;
  background: linear-gradient(
    45deg,
    #00F0FF, #FF00FF, #FFE600, #00F0FF
  );
  background-size: 400% 400%;
  animation: gradient-shift 3s ease infinite;
  z-index: -1;
}

@keyframes gradient-shift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

DO'S & DON'TS

✓DO'S

  • 01

    Use neon colors sparingly — they're power, not wallpaper

  • 02

    Layer text-shadows for realistic neon glow effect

  • 03

    Combine cyan, magenta, and yellow for classic triad

  • 04

    Use scanlines and glitch effects on interactive elements

  • 05

    Keep backgrounds dark (#0D0D0D minimum)

✗DON'TS

  • 01

    Don't overuse glow effects — they lose impact

  • 02

    Avoid light backgrounds that kill the neon aesthetic

  • 03

    Don't use too many different neon colors at once

  • 04

    Avoid subtle animations — cyberpunk is bold and in-your-face

  • 05

    Don't make glitch effects too slow — they should feel broken
